## 💡 技術特點

### 1. 佈局管理
```
QVBoxLayout (主佈局)
├─ 標題標籤
├─ 副標題標籤
├─ 垂直間隔 (20px)
├─ 輸入分組框
│  └─ QVBoxLayout
│     ├─ QHBoxLayout (名稱列)
│     │  ├─ 標籤
│     │  └─ 輸入框
│     └─ QHBoxLayout (描述列)
│        ├─ 標籤
│        └─ 文字區域
├─ 垂直間隔 (可擴展)
└─ QHBoxLayout (按鈕列)
   ├─ 水平間隔
   ├─ 執行按鈕
   ├─ 處理按鈕
   ├─ 重置按鈕
   └─ 水平間隔
```

        this, &MainWindow::onActionButtonClicked);
