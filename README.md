# MidtermAI

## 專案簡介
MidtermAI 是一個基於 Qt 框架開發的智能應用系統，提供現代化的使用者介面和高效的 AI 解決方案。

## 最新更新 - UI 全面優化 🎨

本專案已完成使用者介面的全面優化，包括：

### ✨ 主要特色
- **現代化設計**: 採用 Material Design 風格，提供專業且美觀的介面
- **完整功能**: 表單輸入、資料處理、狀態回饋等完整功能
- **智能驗證**: 輸入驗證和錯誤處理機制
- **豐富色彩**: 藍色、綠色、紅色的語意化色彩系統
- **響應式佈局**: 使用 Qt 佈局管理器實現靈活的介面
- **快捷鍵支援**: 完整的選單系統和鍵盤快捷鍵

### 📋 功能清單
- 標題與副標題顯示
- 名稱輸入欄位
- 多行描述文字區域
- 三個功能按鈕（執行、處理、重置）
- 完整選單系統（檔案、編輯、說明）
- 即時狀態列訊息
- 輸入驗證和錯誤提示

### 🎨 設計亮點
- Material Design 色彩系統
- 圓角按鈕和輸入框
- 懸停和焦點狀態效果
- 清晰的視覺層次
- 適當的間距和對齊

### 開發環境

## 建置與執行

### 使用 Qt Creator
1. 開啟 `MidtermAI.pro` 專案檔
2. 設定建置環境
3. 點擊「建置」按鈕
4. 點擊「執行」按鈕

### 使用命令列
```bash
qmake MidtermAI.pro
make
./MidtermAI
```

## 使用說明

### 基本操作
1. **輸入名稱**: 在「名稱」欄位輸入您的資料名稱
2. **輸入描述**: 在「描述」區域輸入詳細說明
3. **執行操作**: 點擊綠色「執行操作」按鈕
4. **處理資料**: 點擊藍色「處理資料」按鈕來處理輸入的資料
5. **重置**: 點擊紅色「重置」按鈕清除所有輸入

### 快捷鍵
- `Ctrl+N`: 新增
- `Ctrl+O`: 開啟
- `Ctrl+S`: 儲存
- `Ctrl+Q`: 結束程式
- `Ctrl+X`: 剪下
- `Ctrl+C`: 複製
- `Ctrl+V`: 貼上

### 選單功能
- **檔案選單**: 提供新增、開啟、儲存和結束功能
- **編輯選單**: 提供基本的剪下、複製、貼上功能
- **說明選單**: 顯示應用程式的版本和資訊
