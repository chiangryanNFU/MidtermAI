# MidtermAI UI 優化說明文檔

## 概述
本文檔說明 MidtermAI 應用程式的使用者介面優化內容，包括元件類型、功能、色彩配置和版面設計。

## 優化內容

### 1. 元件類型改進

#### 原始介面
- 單一按鈕 (80x30 px)
- 固定位置配置
- 無標題或說明

#### 優化後介面
- **標題標籤** (`QLabel`): 大型、醒目的應用程式標題
- **副標題標籤** (`QLabel`): 應用程式說明文字
- **分組框** (`QGroupBox`): 將相關輸入元件組織在一起
- **文字輸入框** (`QLineEdit`): 用於單行文字輸入，包含佔位符文字
- **多行文字編輯器** (`QTextEdit`): 用於多行描述輸入，限制高度為 80px
- **三個功能按鈕** (`QPushButton`):
  - 執行操作按鈕 (綠色)
  - 處理資料按鈕 (藍色)
  - 重置按鈕 (紅色)
- **選單欄** (`QMenuBar`): 檔案、編輯、說明選單
- **狀態列** (`QStatusBar`): 顯示操作狀態訊息

### 2. 功能增強

#### 新增功能
1. **輸入驗證**: 檢查必填欄位是否填寫
2. **狀態回饋**: 透過狀態列即時顯示操作狀態
3. **對話框提示**: 使用 QMessageBox 提供使用者回饋
4. **重置功能**: 一鍵清除所有輸入欄位
5. **選單系統**: 完整的選單結構，包含快捷鍵支援
6. **關於對話框**: 顯示應用程式版本和資訊

#### 快捷鍵支援
- `Ctrl+N`: 新增
- `Ctrl+O`: 開啟
- `Ctrl+S`: 儲存
- `Ctrl+Q`: 結束
- `Ctrl+X`: 剪下
- `Ctrl+C`: 複製
- `Ctrl+V`: 貼上

### 3. 色彩配置

採用 Material Design 色彩系統，提供現代且專業的視覺體驗：

#### 主要色彩
- **主題藍色**: `#2196F3` (按鈕預設顏色)
  - 懸停: `#1976D2`
  - 按下: `#0D47A1`
- **成功綠色**: `#4CAF50` (執行操作按鈕)
  - 懸停: `#45a049`
- **警告紅色**: `#f44336` (重置按鈕)
  - 懸停: `#da190b`

#### 中性色彩
- **背景色**: `#f5f5f5` (淺灰色，主視窗背景)
- **內容背景**: `#ffffff` (白色，中央區域)
- **邊框色**: `#e0e0e0` (淺灰色)
- **標題色**: `#1976D2` (深藍色)
- **次要文字**: `#666666` (中灰色)

#### 互動狀態
- **焦點狀態**: 藍色邊框 (`#2196F3`)
- **懸停效果**: 顏色加深
- **按下效果**: 顏色最深

### 4. 版面設計

#### 佈局系統
採用 Qt 的佈局管理器，實現響應式和彈性的介面：

1. **主要垂直佈局** (`QVBoxLayout`):
   - 邊距: 20px (上下左右)
   - 元件間距: 15px
   - 自動調整元件大小

2. **水平佈局** (`QHBoxLayout`):
   - 用於標籤和輸入欄位的配對
   - 用於按鈕列，包含彈性空間

3. **間隔器** (`QSpacer`):
   - 垂直間隔: 在主要區塊之間創建視覺分隔
   - 水平間隔: 在按鈕兩側創建對稱空間

#### 視覺層次
1. **頂部**: 標題和副標題 (居中對齊)
2. **中部**: 輸入區域分組框
3. **底部**: 操作按鈕列 (居中對齊，左右留空)

#### 尺寸規格
- 視窗大小: 700x500 px
- 按鈕最小尺寸: 120x40 px
- 多行文字最大高度: 80 px
- 標籤最小寬度: 60 px

### 5. 樣式表詳細說明

#### QPushButton 樣式
```css
- 背景色: 根據按鈕類型不同
- 文字色: 白色
- 無邊框
- 圓角: 6px
- 內距: 10px 20px
- 字體大小: 14px
- 字體粗細: 粗體
- 最小高度: 36px
```

#### QLineEdit 和 QTextEdit 樣式
```css
- 邊框: 2px 實線 #e0e0e0
- 圓角: 4px
- 內距: 8px
- 字體大小: 13px
- 背景色: 白色
- 焦點邊框: 2px 實線 #2196F3
```

#### QGroupBox 樣式
```css
- 邊框: 2px 實線 #e0e0e0
- 圓角: 6px
- 上邊距: 10px
- 字體粗細: 粗體
- 上內距: 15px
- 標題色: #1976D2
```
